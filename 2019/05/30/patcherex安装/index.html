前言

patcherex是一个Shellphish团队开源的二进制补丁的工具
https://github.com/angr/patcherex

安装

使用git clone https://github.com/angr/patcherex.git下载到本地

在安装过程中可能会出现缺少的模块可以自行下载安装
如下

运行示例报错

‘’’
patcherex.errors.CompilationError: File: /tmp/CADET_00003b61zdk42.s Error: (b’’, b’/home/tianhe/Desktop/pwn_tools/patcherex/Other_python_package/compilerex/compilerex/scripts/../bin/clang: error while loading shared libraries: libtinfo.so.5: cannot open shared object file: No such file or directory\n’

solve:apt-get install lib32ncurses-dev

error while loading shared libraries: libz.so.1: cannot open shared object file

solve:apt-get install lib32z1
‘’’
